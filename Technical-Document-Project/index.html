Introduction

Hello and welcome to our very basic introduction to HTML!


HTML stands for Hyper-Text-Markup-Language. This is the standard markup language for creating web pages. It describes the structure of the web page, it consists of elements, and it tells the browser how to display content. HTML is a great place to begin your programming journey. It teaches you structure and gives you an understanding of how and why certain code works! A great free resource is freeCodeCamp.org.


Here are some reasons to learn HTMl:

  • Creating a website for your business.
  • HTML foundation helps with learning other pogramming languages.
  • Looking to change your career.
  • You will understand how websites work.
  • It's free to learn!

What is an element? An element is defined by an opening tag, the content, and the closing tag.

<tagname>content of tag...</tagname>

Note that some elements are self-closing, which means they do not require another closing tag.



Why Learn HTML

Like previously mentioned above, HTML (and CSS), can be a great foundation for learning how to code. Learning HTML might also be useful for personal projects, e-commerce websites, etc. If you ever need a website, you will know how to make things function and how to style things.

What is CSS? CSS stands for Cascading Style Sheets. This is where all of the flashy parts of a website come from.



Setting Up HTML File

In order for our code to work, we must first create a .html file. You can even do this with notepad, though working in dedicated programming software makes things a lot easier. Now we begin entering in our HTML code. On the very first line you want to make sure that you put <!DOCTYPE html>. This declaration tells the browser what type of document to expect, and what to do with the information within that document.


We should have a basic page that looks like this.

<!DOCTYPE html>

Now we need an HTML element. This is the root element. All the code that we will be writing will be nested in this element.


<!DOCTYPE html>
<html>

</html>

Next we need to add a <head> element. This is where meta data about the HTML page lives. We will also add a <title> element to our code. This element specifies a title for the HTML page that will show in the Tab bar of your browser.

<!DOCTYPE html>
<html>

<head>
<title> Hello World! </title>
</head>

</html>

Now we need to add a <body> element to our web page and fill it. A <body> element is the container for all visible content. This includes headings, paragraphs, images, links, tables, lists, and many more!

<!DOCTYPE html>
<html>

<head>
<title> Hello World! </title>
</head>
<body>

<h1>Hello World!</h1>
<p>You coded HTML!</p>

</html>


CSS

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are to be displayed on screen, paper, or in other media. This is what we will use to style our web page.


Originally we created an .html file. Now we need to create a CSS file, and link it to our HTML file. We should first create a styles.css file in the same folder as our HTML file.


Now we should link it to our HTML file. We do this by using the <link> element. Please note that a <link> element is self closing.


<!DOCTYPE html>
<html>

<head>
<link rel="stylesheet" href="styles.css"/>

<title> Hello World! </title>

</head>
<body>

<h1>Hello World!</h1>
<p>You coded HTML!</p>

</html>

We won't be going too in depth about CSS, so if you would like to learn more, please visit freeCodeCamp.org to continue in your programming journey!


Now open your styles.css file. We will be changing the color of our <h1> element to Blue. To do this, we put the following.


h1 {
color: Blue;
}

Remember to indent the "color: Blue;" line.
